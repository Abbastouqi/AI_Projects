\section{Executive Summary}
% =============================================================================

The \textbf{AI PC Agent} is an intelligent desktop automation system designed to understand and execute user commands through both voice and text inputs. This comprehensive report documents the development, architecture, features, and testing results of the system.

The agent successfully automates desktop tasks including:
\begin{itemize}
    \item Opening and launching applications
    \item Creating and managing documents
    \item Web searching and browsing automation
    \item University admission application automation
    \item User profile management and auto-fill functionality
    \item Policy and information retrieval
\end{itemize}

\subsection{Intent Matching Algorithm}

\begin{lstlisting}[language=Python]
def parse(self, text: str) -> Dict[str, Any]:
    """Parse natural language command into structured intent"""
    text_lower = text.lower()
    
    # Score each intent based on keyword matches
    intent_scores = {}
    for intent_name, pattern in self.intent_patterns.items():
        score = 0
        for keyword in pattern['keywords']:
            if keyword in text_lower:
                score += len(keyword.split())
        
        if score > 0:
            intent_scores[intent_name] = score
    
    # Select best matching intent
    best_intent = max(intent_scores, key=intent_scores.get)
    confidence = min(intent_scores[best_intent] / 3, 1.0)
    
    return {
        'intent': best_intent,
        'confidence': confidence,
        'parameters': parameters
    }
\end{lstlisting}

\section{Troubleshooting Guide}
% =============================================================================

\subsection{Common Issues}

\subsubsection{Issue: Audio Input Not Working}

\textbf{Solution:}
\begin{itemize}
    \item Check microphone is connected
    \item Verify in system settings
    \item Re-calibrate: Run agent and wait for calibration
    \item Update audio drivers
\end{itemize}

\subsubsection{Issue: WebDriver Initialization Fails}

\textbf{Solution:}
\begin{itemize}
    \item Ensure Chrome is installed
    \item Update ChromeDriver: \texttt{pip install --upgrade webdriver-manager}
    \item Clear cache: Delete \texttt{.wdm} folder
\end{itemize}

\subsubsection{Issue: Command Not Recognized}

\textbf{Solution:}
\begin{itemize}
    \item Check command syntax
    \item Verify keywords in intent parser
    \item Try similar command variations
    \item Check logs for intent detection score
\end{itemize}

\subsubsection{Issue: Form Auto-fill Fails}

\textbf{Solution:}
\begin{itemize}
