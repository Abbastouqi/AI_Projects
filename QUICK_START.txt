INSTALLATION
------------
1. pip install -r requirements.txt
2. Done!


RUN THE AGENT
-------------
Option 1: Web Interface (Recommended)
   python web_server.py
   Open: http://localhost:5000

Option 2: Desktop GUI
   python main.py


MAIN COMMANDS
-------------
Auto-Fill Forms:
   "auto fill"                    # Automatically fill entire form!
   "autofill"                     # Quick version
   "fill this form"               # Natural language

Web Browsing:
   "open google.com"              # Open any website
   "search Python tutorials"      # Google search

Open Applications:
   "open calculator"              # Open calculator
   "open notepad"                 # Open notepad
   "open chrome"                  # Open Chrome

Manual Form Filling:
   "fill name with John Doe"      # Fill specific field
   "fill email with john@email.com"
   "click submit"                 # Submit form
   "press enter"                  # Press Enter

System Control:
   "shutdown computer"            # Shutdown PC
   "restart computer"             # Restart PC
   "open downloads"               # Open downloads folder

TROUBLESHOOTING
---------------
Voice not working?
   - Check microphone permissions
   - Use Chrome or Edge browser

Form not filling?
   - Wait for page to load
   - Make sure form is visible
   - Try: "fill name with John"

App won't open?
   - Check app is installed
   - Run as Administrator
