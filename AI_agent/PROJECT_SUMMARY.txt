PROJECT NAME
------------
AI Agent - PC Automation Assistant


DESCRIPTION
-----------
Voice and text-based AI agent that can perform ANY task on your PC.
Main feature: Automatic form filling on ANY website with one command!

✓ Auto-fill forms (NEW!)        - "auto fill" fills entire forms automatically

AUTO-FILL FEATURE (NEW!)
------------------------
Automatically detects and fills forms on ANY website!

Commands:
   - "auto fill"
   - "autofill"
   - "fill this form"
   - "smart fill"

How it works:
   1. Scans page for forms
   2. Detects all input fields
   3. Matches fields to data (name, email, phone, etc.)
   4. Fills fields automatically
   5. Shows detailed results

Supported fields:
   - Name (name, full name, username)
   - Email (email, e-mail, mail)
   - Phone (phone, mobile, telephone)
   - Address (address, street, location)
   - City, Country
   - Message, Subject
   - Company, Website
   - And more!

Detection strategies:
   1. By name attribute
   2. By id attribute
   3. By placeholder text
   4. By label text
   5. By field type

-------------
File: config.json

   - selenium_headless: false (show browser)
   - voice_enabled: true (enable voice input)
   - tts_enabled: true (enable voice output)
   - log_level: "INFO" (logging level)
